## Description
MView is a RPG Maker MV resource editor.

## Features
 * Multiple files and folders loadable.
 * **(MV)** RPGMVO, RPGMVM, RPGMVW, RPGMVP / **(MZ)** ogg_, m4a_, wav_, png_ files cryptography functions supported.
